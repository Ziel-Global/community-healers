# Soft Skill Training Portal

Governance and Certifications for Soft Skill Training.

## Project info

This project manages training centers, candidates, and certification processes.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm i
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Query
- React Router

## Features

- Super Admin Dashboard with participation trends
- Center management and location tracking
- Candidate registration and training scheduling
- Audit logging for system security
